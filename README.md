# romia.dev

Personal portfolio of Abdelrahman Abouroumia (Romia), AI Engineer and co-founder of Zaylon AI. Built with Next.js (App Router), React, Tailwind CSS, and Motion.

Live at [romia.dev](https://romia.dev).

## Development

```bash
npm install
npm run dev      # start the dev server on http://localhost:3000
npm run build    # production build
npm run start    # serve the production build
npm run lint     # lint
```

## Structure

- `src/app` : routes, layout, metadata, dynamic OG images, project case studies
- `src/components/sections` : homepage sections
- `src/data/resume.ts` : all content (personal info, projects, experience, skills)
- `abdelrahman_abouroumia_cv.tex` : LaTeX source for the downloadable CV
